# 積立NISAを維持しつつ信越ポリマー・ソニー重点へ移行する計画

**作成日**: 2026-06-30

**目的**: FANG+積立を維持しながら、直接メモリ株偏重を落とし、信越ポリマーとソニーを将来ポートフォリオの重点枠へ移行する。

## 1. 中核方針

月10万円のつみたて投資枠は、長期成長枠として維持する。FANG+は現時点でユーザーの成長仮説と最も合致する投資信託であり、単純に停止しない。

一方で、FANG+は10銘柄集中型であり、MUや将来のSPCX採用により、個別株側のMU/SNDK/SPCX保有とリスクが重複する。したがって、積立は攻め枠として残し、個別株側で直接メモリ株を削減する。

優先順位は以下とする。

1. 機動用現金・MMFを200万円以上に維持する。
2. MU/SNDK/SPCXの個別買い増しを停止する。
3. 戻り局面でMU特定、SNDK、SPCXを売却候補にする。
4. 信越ポリマーを最大20%までの重点候補にする。
5. ソニーを10-20%の中核候補にする。
6. 任天堂と上流半導体は第2段階の候補にする。

## 2. 目標配分

| 区分 | 目標・上限 | 位置づけ |
|---|---:|---|
| 現金・MMF・機動余力 | 200万円以上 | デイトレ、押し目買い、信用余力維持の基盤 |
| 信越ポリマー | 初期10%、条件付き15%、最大20% | 明確な割安株への重点投資枠 |
| ソニー | 10-20% | エンタメIP・センサー・資本配分改善の中核候補 |
| FANG+投信 | 月10万円積立を継続 | つみたて投資枠の攻め枠 |
| iFreeNEXT NASDAQ100 | 条件付きサブ候補 | FANG+のSPCX集中が強まった場合の分散先 |
| 任天堂 | 5-10%候補 | Switch 2継続確認後のIP枠 |
| 上流半導体 | 10-15%候補 | 東京エレクトロン、ASML、KLA、SCREEN等 |
| 直接メモリ株 | 縮小対象 | MU/SNDKの新規買い増しは禁止 |

## 3. 積立NISAルール

### 3.1 FANG+を継続する条件

以下のどちらかを満たす限り、FANG+月10万円を継続する。

- FANG+にSPCXが採用されていない。
- FANG+にSPCXが採用されても、個別株側のSPCXを十分に縮小済みで、全体のSPCX実質比率が過大でない。

### 3.2 NASDAQ100へ一部変更する条件

FANG+にSPCXが正式採用され、等ウェイトに近い10%前後の構成比となる場合は、以降の積立を一時的に以下へ変更する。

| 積立先 | 月額 |
|---|---:|
| iFreeNEXT FANG+インデックス | 5万円 |
| iFreeNEXT NASDAQ100インデックス | 5万円 |

NASDAQ100はFANG+より分散されるが、成長率の源泉は大型テック依存が大きい。FANG+の完全代替ではなく、重複緩和用のサブ候補として扱う。

### 3.3 個別株との重複管理

FANG+を継続する代わりに、以下の個別株買い増しは禁止する。

- MU
- SNDK
- SPCX
- SMCI

MUとSPCXは、FANG+/NASDAQ100経由での間接保有を含めて実質比率を確認する。過大な場合は、個別株側を先に削減する。

## 4. 個別株の実行ルール

### 4.1 売却原資

| 対象 | 行動 | 理由 |
|---|---|---|
| MU 特定口座 | 取得単価近辺への戻りで全売却候補 | 税務上動かしやすく、直接メモリ偏重を大きく下げられる |
| SNDK NISA | 取得単価近辺または強い反発局面で売却候補 | NANDサイクル依存が強く、FANG+積立と別に持つ必要性が低い |
| SPCX NISA | 1-2株売却候補 | FANG+/NASDAQ100採用時の重複リスクを抑える |
| MU NISA | 最後の調整弁 | 余力200万円を割る場合のみ、5株から3株へ縮小検討 |

### 4.2 信越ポリマー

信越ポリマーは、最大20%まで許容する重点枠とする。ただし小型株、親子上場、流動性リスクがあるため一括で20%にしない。

| 価格・条件 | 行動 |
|---|---|
| 2,400円台 | 初期10%まで許容 |
| 2,000円前後以下 | 15-20%まで段階買いを検討 |
| 3,200円以上 | 新規買い停止、一部利益確定を検討 |
| 精密成形品利益率10%以下 | 仮説撤回・売却検討 |
| 信越化学が支配権を放棄 | TOB仮説崩壊として売却検討 |

根拠は、同社レポートにおける以下の評価に置く。

- FY2026売上高1,151億円、営業利益140億円、親会社当期純利益99億円。
- 自己資本比率84.4%、広義ネットキャッシュ474億円。
- 精密成形品が利益構成比72.8%。
- 基準シナリオの10年後目標株価3,969.8円、悲観シナリオ1,477.8円。

### 4.3 ソニー

ソニーは、信越ポリマーの初期ポジション構築後に積み上げる中核候補とする。

| 価格・条件 | 行動 |
|---|---|
| 3,268円近辺 | 買い候補 |
| 3,000円割れ | 優先的に積み増し検討 |
| 3年基準シナリオ3,675円接近 | 新規買いは慎重化 |
| KADOKAWA関連の巨額資本流出 | 再評価 |
| ゲーム利益率の悪化 | 再評価 |
| センサー利益率の低下 | 再評価 |

根拠は、ソニーレポートにおける以下の評価に置く。

- 金融スピンオフ後の「ソニー2.0」への純化。
- 継続事業FCF約1.07兆円、FCFコンバージョン率103.3%。
- 実質EV/EBITDA 6.45倍、FCF利回り5.51%。
- 音楽、アニメIP、積層型CMOSセンサーの複合モート。
- 3年後基準シナリオ株価3,675.4円、楽観シナリオ5,298.9円。

## 5. レバレッジ・リスク管理

レバレッジは、明確な割安株に限定する。対象は原則として、信越ポリマーとソニーのように一次資料ベースで安全域が確認済みの銘柄に限る。

以下へのレバレッジは禁止する。

- MU
- SNDK
- SPCX
- SMCI
- FANG+
- NASDAQ100

レバレッジ込みでも、機動用現金・MMF200万円を割らない。200万円を割る場合は、新規買付よりもポジション縮小を優先する。

---

## 6. 月次レビュー項目

毎月末に以下を確認する。

| 項目 | 判定基準 |
|---|---|
| 機動余力 | 200万円以上を維持 |
| 信越ポリマー比率 | 20%以下 |
| ソニー比率 | 20%以下 |
| FANG+構成 | SPCX採用有無、MU比率 |
| NASDAQ100構成 | SPCX採用後の実質比率 |
| 直接MU + FANG+経由MU | 過大なら直接MUを削減 |
| 直接SPCX + 投信経由SPCX | 採用後に再計算 |
| メモリ株 | 新規買い増しなし |

## 7. 決算レビュー項目

### 信越ポリマー

- 精密成形品部門の利益率。
- 半導体容器需要。
- 自己資本比率とネットキャッシュ。
- 親会社である信越化学の保有方針。
- 中期経営計画に対する進捗。

### ソニー

- 継続事業FCF。
- 自社株買いと株主還元。
- ゲーム事業の利益率。
- 音楽・アニメ事業の成長率。
- イメージセンサーの利益率とApple依存。
- KADOKAWA関連の追加資本流出。

### MU/SNDK

- 取得単価近辺に戻ったか。
- 売却前提を覆すほどの構造変化があるか。
- HBM/NANDの価格サイクルがピークアウトしていないか。

---

## 8. 実行順序

1. FANG+月10万円積立は継続する。
2. MU/SNDK/SPCX/SMCIの個別買い増しを停止する。
3. MU特定、SNDK、SPCXの戻り売りで機動余力200万円を確保する。
4. 信越ポリマーを10%まで構築する。
5. 信越ポリマーが2,000円前後以下へ下がる場合、15-20%まで段階的に増やす。
6. ソニーを10%まで構築し、価格条件と決算確認で20%まで増やす。
7. FANG+へのSPCX採用が確認されたら、積立をFANG+ 5万円 / NASDAQ100 5万円へ一時変更するか再判定する。
8. 任天堂と上流半導体は、上記が整った後の第2段階として検討する。

---

## 9. 注意事項

本計画は学習・研究目的のポートフォリオ運用メモであり、投資助言ではない。実際の売買は、証券口座上の税制、NISA枠、為替、信用余力、手数料、生活資金を確認したうえで自己責任で行う。
