# Japan Market Adaptation Specification

- 文書ID: `ABG-MARKET-JP-SPEC`
- 上位仕様: `docs/global-market-adaptation/spec.md` (`ABG-MARKET-ADAPTATION-SPEC`)
- 状態: Draft

## 1. 概要

本仕様書は、日本（JP）市場を対象とする `JapanMarketAdapter` のデータ取得経路、対象書類、固有分析論点、および村上世彰型・藤野英人型レンズの統合仕様を定義する。

## 2. データ取得経路

- **主要正本**: EDINET API v2 (`EDINET_API_KEY` 任意設定), TDnet Web, JPX上場会社情報サービス, コーポレートガバナンス報告書, 会社公式IR (決算説明資料, 統合報告書, 中計)
- **二次情報**: Yahoo Finance, 金融報道

## 3. 日本固有チェック項目 & 投資家レンズ統合

1. **村上世彰型レンズ**:
   - 余剰現金・ネットキャッシュの算出
   - 政策保有株式の規模および縮減方針
   - 非中核事業・非事業用不動産の特定
   - 親子上場および支配株主との利益相反
   - 資本コストや株価を意識した経営（PBR 1倍割れ改善計画）の実効性
2. **藤野英人型レンズ**:
   - 経営者の誠実性・過去の公約履行状況
   - 組織能力、企業文化、権限移譲、人的資本投資
   - 後継者問題および経営継続性
   - 中小型成長企業における定性成長余地
