**作成日**: 2026-06-30

**目的**: FANG+積立を維持しながら、直接メモリ株偏重を落とし、信越ポリマーとソニーを将来ポートフォリオの重点枠へ移行する。

## 1. 中核方針

月10万円のつみたて投資枠は、長期成長枠として維持する。FANG+は「10社すべてを同じ強度で高評価しているから買う商品」ではない。つみたて投資枠で購入可能な投資信託の中で、資本を高い成長性と再投資能力を持つ企業へ比較的集中して配分できる手段として選択する。

判断基準は、売上成長率そのものではなく、企業が受け取った資本を将来の利益、FCF、1株当たり価値へ効率よく変換できるかである。成熟企業であっても、既存事業から生じる余剰資金をAI、クラウド、半導体、データ基盤、その他の成長分野へ高い収益率で再投資できる限り、FANG+内での保有を許容する。

役割分担は次の通りに固定する。FANG+はつみたて投資枠を使った成長資本配分、MU特定口座は次の主要DRAM需給イベントまでの資金集中を取る戦術枠、MU/SNDKのNISAは2026年中の上昇を非課税で取る期限付き成長枠、SPCXは上限付きの長期オプション、信越ポリマーとソニーは恒久的な中核候補、現金・MMFは機動余力である。

優先順位は以下とする。

1. 未拘束の現金・MMFを200万円以上に維持する。
2. MU/SNDK/SPCXの個別買い増しを停止する。
3. MU特定はイベント前の戦術売却候補、MU/SNDKのNISA分は2026年末を基本期限とする期限付き成長投資にする。
4. 信越ポリマーとソニーは固定順序で買わず、各時点の安全域、期待収益率、資金余力を比較し、条件を先に満たした方から段階取得する。
5. SPCXは実質合算比率5%目安、10%上限の長期オプション枠として保有する。
6. 任天堂と上流半導体は第2段階の候補にする。

## 2. 目標配分

すべての比率は、現金・MMF、投資信託、国内株、外国株を含む総金融資産の時価を分母とする。外国資産は月末の基準為替レートで円換算する。現金・MMF200万円は信用取引の担保に算入しない未拘束資金とし、この金額を維持できない場合は新規購入しない。

| 区分 | 目標・上限 | 位置づけ |
|---|---:|---|
| 現金・MMF・機動余力 | 200万円以上 | デイトレ、押し目買い、信用余力維持の基盤。ただし信用担保には算入しない |
| 信越ポリマー | 初期許容上限10%、条件付き15%、最大20% | 明確な割安株への重点投資枠。10%は買付義務ではない |
| ソニー | 10-20% | エンタメIP・センサー・資本配分改善の中核候補 |
| FANG+投信 | 月10万円積立を継続 | つみたて投資枠の攻め枠 |
| iFreeNEXT NASDAQ100 | 条件付きサブ候補 | FANG+の成長配分効率が低下した場合の代替・分散先 |
| 任天堂 | 5-10%候補 | Switch 2継続確認後のIP枠 |
| 上流半導体 | 10-15%候補 | 東京エレクトロン、ASML、KLA、SCREEN等 |
| SPCX | 実質合算比率5%目安、10%上限 | SpaceX/Starlinkの長期オプション枠 |
| 直接メモリ株 | 新規買い増し禁止 | MU特定は戦術枠、MU/SNDK NISAは2026年期限付き成長枠 |

## 3. 積立NISAルール

### 3.1 FANG+を継続する条件

以下を満たす限り、FANG+月10万円を継続する。

- FANG+が、つみたて投資枠内で成長企業への高濃度配分手段として最も有効である。
- 直接成長産業と、高効率で成長分野へ再投資する成熟企業の合計が、おおむね70-80%以上である。
- 構成企業のROIC、FCF成長、株式報酬控除後の1株当たりFCF、自社株買いによる実株数減少が劣化していない。
- SPCXが採用された場合でも、SPCX実質合算比率が8%未満に収まっている。

### 3.2 NASDAQ100へ一部変更する条件

SPCXのFANG+採用だけを理由に、NASDAQ100へ自動分割しない。NASDAQ100へ一部変更するのは、以下のいずれかが起きた場合に限る。

- SPCX実質合算比率が8%に達し、新規の間接配分を含めた再確認が必要になる。
- FANG+の成長産業・高効率再投資企業比率が70%を下回る。
- FANG+構成銘柄の再投資効率が悪化し、株式報酬控除後の1株当たりFCF成長が鈍化する。
- FANG+が過去の大型勝者に固定され、新しい成長企業を取り込めない。

変更する場合も、FANG+を自動的にゼロにするのではなく、以後の積立分だけを `FANG+ 5万円 / NASDAQ100 5万円` などへ一時変更する。

---

### 3.3 FANG+の四半期レビュー

FANG+は、四半期または半年ごとに以下で評価する。

| 評価項目 | 継続基準 |
|---|---|
| 成長産業への実質配分 | 直接成長産業 + 高効率再投資企業が70-80%以上 |
| 再投資効率 | ROIC、FCF成長、株式報酬控除後の1株当たりFCFが健全 |
| 指数の新陳代謝 | SPCXのような新しい大規模成長企業を適時取り込める |
| 期待収益率 | 1株当たり利益成長率 + 株主還元率 +/- 評価倍率変化が十分 |

### 3.4 個別株との重複管理

MUとSPCXは、FANG+/NASDAQ100経由での間接保有を含めて実質比率を管理する。

- 直接MU + FANG+経由MUが過大なら、直接MUを優先的に削減する。
- SPCXは常に実質合算比率で管理する。SPCX実質比率 = (直接SPCX時価 + FANG+残高 x FANG+内SPCX比率 + NASDAQ100残高 x NASDAQ100内SPCX比率) / 総金融資産。
- SPCX実質比率が8%未満なら継続保有、8-10%なら新規の間接配分を含めて再確認、10%超なら直接SPCXの一部利確を検討する。事業仮説が崩壊した場合は比率に関係なく売却再判定する。

## 4. 個別株の実行ルール

### 4.1 米国メモリ株の出口ルール

MU/SNDKは単純な「縮小対象」ではなく、2027年末までの需要・収益可視性を根拠に、2026年中の価格上昇を非課税で取りに行く期限付き成長投資として扱う。永久保有の中核ではないが、取得価格へ戻っただけで売る対象でもない。

この前提では、口座ごとに役割を分ける。

| 保有 | 位置づけ | 基本方針 |
|---|---|---|
| MU特定口座3株 | 戦術的な上乗せ枠 | 次の主要DRAM需給イベントまで資金集中を取り、イベント前に機械的に決済する |
| MU NISA5株 | 期限付き成長コア | 2026年末を基本期限に保有し、2027年需要の織り込み進展または需給悪化で売却する |
| SNDK NISA1株 | 期限付き高ベータ成長枠 | 2026年末を基本期限に保有し、NAND・エンタープライズSSD需要、期待過熱、需給悪化で判断する |
| SPCX NISA | 戦略的オプション | SpaceX/Starlinkの独自性を長期オプションとして保持する |

MU特定口座の予定イベントTは、原則として次のうち最も早いものとする。SNDKやキオクシアの決算はNAND確認イベントであり、MU特定口座を機械的に決済する基準にはせず補助情報として扱う。

- Micron決算
- Samsungの業績速報または本決算
- SK hynix決算

MU特定口座の執行ルール:

- T-10営業日: SOX、主要メモリ株、MUの相対強度と出来高の監視を開始する。
- T-5営業日: 決済を開始する。取得単価への回復は条件にしない。
- T-3営業日: 原則として全3株の決済を完了する。3株しかないため、T-4またはT-3の一括売却でもよい。
- T-2営業日: 絶対的な最終期限とし、未決済分を残さない。
- T-5より前でも、SOXやメモリ株全体に対する相対的弱さが明確になれば前倒し決済する。

突発イベントは日程基準ではなく、発表時点で即時再判定する。対象は、契約価格引き下げ、増産・稼働率引き上げ、顧客契約の取消し、在庫急増、輸出規制や政策の重大変更である。

NISA分は通常の四半期決算を機械的に避けない。NISAは非課税の上昇余地を取りに行く枠であり、毎回の決算前に閉じると上昇の主要部分を取り逃がす可能性がある。NISA分をイベント前に売るのは、需給仮説が悪化した場合、期待が過熱して2027年利益を十分以上に織り込んだ場合、またはポートフォリオ比率が過大になった場合に限る。

2026年10-12月を最終レビュー期間とし、原則として2026年末までに出口を決める。保有を延長する場合も恒久保有とはせず、次の決算または3-6か月後までの新たな期限付き投資として再設定する。延長条件は、2027年需要が契約・前払い・ガイダンスでさらに強化されたこと、供給増加が需要増加を上回る兆候がないこと、NISA非課税効果を含む期待収益率がソニー・信越ポリマーより依然高いことの3つを満たす場合に限定する。

売却シグナルは強度を分ける。ハードシグナルは1件で前倒し売却、ソフトシグナルは2件で前倒し売却とする。

| 種別 | シグナル | 行動 |
|---|---|---|
| ハード | 契約価格またはASPの明確な下落 | NISA分も前倒し売却 |
| ハード | 在庫の急増または会社計画を上回る積み上がり | NISA分も前倒し売却 |
| ハード | ビット供給成長が需要成長を明確に上回るガイダンス | NISA分も前倒し売却 |
| ハード | 長期契約の取消し・下方再交渉 | NISA分も前倒し売却 |
| ハード | 業績ガイダンスの引き下げ | NISA分も前倒し売却 |
| ソフト | 価格上昇率の鈍化・横ばい化 | 2件で前倒し売却 |
| ソフト | リードタイム短縮 | 2件で前倒し売却 |
| ソフト | 設備投資加速 | 2件で前倒し売却 |
| ソフト | 好材料に対する株価反応の低下 | 特定口座の早期決済には使えるが、NISA売却にはもう1つのシグナルを必要とする |
| ソフト | SOXに対する相対的弱さ | 2件で前倒し売却 |
| ソフト | 顧客の在庫調整コメント | 2件で前倒し売却 |

取得価格、含み損益、直近高値は売却判断の主基準にしない。新規MU/SNDK買い増しは停止し、2026年後半から年末にかけて、メモリ株から信越ポリマー・ソニーなど恒久的中核候補へ資本を移す。

### 4.2 信越ポリマー

信越ポリマーは、最大20%まで許容する重点枠とする。ただし小型株、親子上場、流動性リスクがあるため一括で20%にしない。

| 価格・条件 | 行動 |
|---|---|
| 2,400円台 | 初期10%まで許容 |
| 2,000円前後以下 | 15-20%まで段階買いを検討 |
| 3,200円以上 | 新規買い停止、一部利益確定を検討 |
| 精密成形品利益率10%以下 | 仮説撤回・売却検討 |
| 信越化学が支配権を放棄 | TOB仮説崩壊として売却検討 |

根拠は、同社レポートにおける以下の評価に置く。

- FY2026売上高1,151億円、営業利益140億円、親会社当期純利益99億円。
- 自己資本比率84.4%、広義ネットキャッシュ474億円。
- 精密成形品が利益構成比72.8%。
- 基準シナリオの10年後目標株価3,969.8円、悲観シナリオ1,477.8円。

### 4.3 ソニー

ソニーは、信越ポリマーと並ぶ中核候補とする。固定順序ではなく、その時点の安全域、期待収益率、未拘束現金200万円を維持できるかで段階取得を判断する。

| 価格・条件 | 行動 |
|---|---|
| 3,268円近辺 | 買い候補 |
| 3,000円割れ | 優先的に積み増し検討 |
| 3年基準シナリオ3,675円接近 | 新規買いは慎重化 |
| KADOKAWA関連の巨額資本流出 | 再評価 |
| ゲーム利益率の悪化 | 再評価 |
| センサー利益率の低下 | 再評価 |

根拠は、ソニーレポートにおける以下の評価に置く。

- 金融スピンオフ後の「ソニー2.0」への純化。
- 継続事業FCF約1.07兆円、FCFコンバージョン率103.3%。
- 実質EV/EBITDA 6.45倍、FCF利回り5.51%。
- 音楽、アニメIP、積層型CMOSセンサーの複合モート。
- 3年後基準シナリオ株価3,675.4円、楽観シナリオ5,298.9円。

## 5. レバレッジ・リスク管理

移行完了までは、信越ポリマー・ソニーを含めて新規レバレッジを停止する。現在はリスクオフ方針と機動余力200万円の維持を優先する。

将来レバレッジを再開する場合も、対象は明確な割安株に限定する。対象は原則として、信越ポリマーとソニーのように一次資料ベースで安全域が確認済みの銘柄に限る。

以下へのレバレッジは禁止する。

- MU
- SNDK
- SPCX
- FANG+
- NASDAQ100
- SMCI

現金・MMF200万円は信用取引の担保に算入しない未拘束資金とする。レバレッジ込みでも機動用現金・MMF200万円を割らない、ではなく、200万円を拘束しないことを必須条件にする。

## 8. 実行順序

1. FANG+月10万円積立は継続する。
2. MU/SNDK/SPCX/SMCIの個別買い増しを停止する。
3. MU特定口座3株は、次のMicron、Samsung、SK hynixの予定イベントを基準日Tとし、T-5から決済を開始、T-3を原則完了日、T-2を最終期限にする。
4. 突発的な契約価格引き下げ、増産、契約取消し、在庫急増、政策変更が出た場合は即時再判定する。
5. MU NISA5株は、2026年末を基本期限とする期限付き成長コアとして保有し、ハードシグナル1件、ソフトシグナル2件、評価過熱、期待収益率低下で前倒し売却する。
6. SNDK NISA1株は、2026年末を基本期限とする高ベータ成長枠として保有し、NAND固有の悪化または期待先行の急騰時に売却する。
7. 2026年10-12月を最終レビュー期間とし、原則として2026年末までに出口を決める。延長する場合も恒久保有ではなく、次の決算または3-6か月後までの新たな期限付き投資として再設定する。
8. 信越ポリマーとソニーは固定順序で買わず、安全域、期待収益率、未拘束現金200万円維持を比較して、条件を満たした方から段階取得する。
9. 信越ポリマーは総金融資産の初期10%を許容上限、条件付き15-20%を最大上限とする。
10. ソニーは総金融資産の10-20%を中核候補とする。
11. SPCXは実質合算比率8%で注意、10%超で直接SPCXの一部利確を検討する。
12. FANG+はSPCX採用だけでは分割せず、成長産業比率・再投資効率・実質SPCX比率のいずれかが基準を外れた時だけ積立先を再判定する。
13. 任天堂と上流半導体は、上記が整った後の第2段階として検討する。
