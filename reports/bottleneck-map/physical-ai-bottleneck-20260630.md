## 手順1：mega trend確認

### trend名

Physical AI（フィジカルAI）— 実体世界で知覚・推論・行動するAI搭載機械

### 中核driver

グローバルな労働力不足（製造・物流・建設）とAIソフトウェアの成熟が交差し、人間型・産業用ロボットへの需要が構造的に拡大している。

### 確認済み事象

1. 2026年1-5月：中国ロボティクス企業の資金調達額$5.6B（Crunchbase）、2025年通期の$4.3Bを既に超過
2. 2026年4月：Teslaが2026年CAPEXを$25Bに3倍増、Optimus量産を本格化（NewsBytes）
3. 2026年5月：ロボティクススタートアップの世界調達額が$18.8Bに達し、過去最高を記録（Value Add Pulse / Crunchbase）
4. 2026年3月：McKinseyが人型ロボットサプライチェーン分析を公表、精密減速機・遊星ローラねじ・六軸力覚センサーを3大ボトルネックと特定
5. 2025年12月現在：世界の産業用ロボット設置台数542千台（IFR）、中国が新規設置の74%を占める
6. 2026年1月：Bloombergが「中国はフィジカルAIの巨大実験場」と報道、製造サプライチェーン優位性を指摘
7. 2026年6月：Hyundai Motor、25,000台のAtlas人型ロボット配備計画を発表。Hyundai Mobisが2028年までに年産350,000台のアクチュエータ供給目標を設定

### 年間capital expenditure

世界のロボティクス関連CAPEX（2026年）：推定$250-300B超
- Tesla：$25B（2026年）
- Amazon：$200B（2026年、広義のインフラ含む）
- Google：$175-185B（2026年）
- 中国ロボティクスVC投資：$5.6B（1-5月）
- Goldman Sachs推定：人型ロボット累積投資$50B超（2030年まで）

需給判断：**需要増加率 > 供給能力増加率**

結論：✅ **継続調査** — 複数の独立情報源が一致して大規模成長を確認。物理的制約が明確に存在する。

主な反証条件：
- 人型ロボットの量産が技術的に想定より難航し、2028年以降も万台規模に達しない
- 地政学的緊張の急激な悪化によりサプライチェーンが分断される
- AI投資バブル崩壊による資金調達環境の激変

## 手順2：サプライチェーン分解

```text
Layer 0（end use）：Physical AI — 人型ロボット、産業用ロボット、自律走行搬送機（AMR）、自動化建機
    │
Layer 1（中核component）：アクチュエータ・モーター、AIエッジコンピュート、バッテリー、構造体
    │
Layer 2（subcomponent・material）：
    ├─ 精密減速機（ハーモニック・RV・サイクロイド・遊星）
    ├─ 精密送りねじ（遊星ローラねじ・ボールねじ）
    ├─ 永久磁石（NdFeB焼結磁石）
    ├─ 六軸力覚センサー・触覚センサー
    ├─ エッジAI半導体（GPU・AIアクセラレータ）
    ├─ バッテリーセル・BMS
    ├─ 炭素繊維・アルミ合金・チタン構造材
    └─ アクチュエータ統合モジュール
    │
Layer 3（upstream equipment・raw material）：
    ├─ 希土類原材料（NdPr、Dy、Tb合金・酸化物）
    ├─ 高純度インディウム・ガリウム
    ├─ 精密加工設備（CNC・歯切盤）
    └─ 半導体製造装置
    │
Layer 4（infrastructure）：
    ├─ 電力・データセンター
    ├─ 人材（制御工学・機械設計）
    ├─ 認証・安全規格（ISO 13482など）
    └─ 地政学的許認可（輸出管理・関税）
```

## 主要な結論

1. **Physical AIのサプライチェーンで最も深刻なボトルネックは精密減速機とNdFeB磁石である。** 世界5社未満しかハーモニック減速機を量産できず、NdFeB磁石は中国85%超依存。これらの制約は2028-2030年に本格化する。

2. **現在の上場企業のバリュエーションは、制約の実在と投資価値の間に大きな乖離がある。** 最も純粋なボトルネック企業（Harmonic Drive、Green Harmonic）はPSR 12-127xと極めて高く、成長実績がバリュエーションを大幅に下回る。

3. **JL MAGが最もバランスの取れた候補である。** NdFeB磁石で世界トップ、PSR 5.2xと現実的、売上高成長率16%を達成。ただし、PER 56xは短期的には割高感があり、希土類価格変動リスクと地政学リスクを内包する。

4. **Nabtescoは安全域は大きいが成長期待は小さい。** PSR 1.9xで割安だが、人型ロボット需要がRV減速機に本格的に波及するかは現時点で不確実。

5. **未上場企業（Seenpin, Link-touch等）のIPO動向は継続監視すべきである。** 特に遊星ローラねじのSeenpinは、IPO後に適正なバリュエーションで投資機会を提供する可能性がある。

6. **本分析は特定時点のスナップショットであり、定期的な更新を要する。** 人型ロボットの量産進捗、各社の決算、地政学的変化に応じてランキングは変動する。
