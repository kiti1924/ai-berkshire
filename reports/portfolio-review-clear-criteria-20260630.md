# ポートフォリオ再判定: 積立NISA・信越ポリマー・ソニー重点化

**作成日**: 2026-06-30  
**使用スキル**: `portfolio-review`  
**目的**: FANG+積立を維持しながら、MU/SNDKの直接メモリ偏重を減らし、SPCXは戦略的オプション枠として管理し、信越ポリマーとソニーを重点化する判断を明確な基準で再判定する。

---

## 1. 判定基準

各銘柄・投信を以下の5軸で評価する。

| 軸 | 配点 | 判定内容 |
|---|---:|---|
| 事業・仮説の質 | 30% | 10年後も理解可能で、競争優位が残るか |
| 安全域・割安性 | 25% | 現在価格に対して保守シナリオでも損失耐性があるか |
| ポートフォリオ内の役割 | 20% | 他保有との重複が少なく、全体のリスク調整後リターンを改善するか |
| 資金拘束・流動性 | 15% | 余力200万円を守りながら保有できるか |
| トリガー明確性 | 10% | 買い増し、維持、縮小、撤退条件が明確か |

総合判定は以下とする。

| スコア | 判定 | 行動 |
|---:|---|---|
| 4.2以上 | Core Buy | 中核候補として積み上げる |
| 3.6-4.1 | Buy / Hold | 条件付きで買い増し、または維持 |
| 3.0-3.5 | Watch / Hold | 監視または少額維持 |
| 2.4-2.9 | Reduce | 反発局面で縮小 |
| 2.3以下 | Exit / Trim | 売却または最小限まで縮小 |

## 4. 結論

### 最優先: 信越ポリマー

信越ポリマーは、現在価格2,400円台で初期10%まで許容できる。最大20%まで認めるが、これは一括ではなく段階的に行う。

実行条件:

- 2,400円台: 初期10%まで構築。
- 2,000円前後以下: 15-20%まで段階買い。
- 3,200円以上: 新規買い停止、一部利確検討。
- 精密成形品利益率10%以下: 仮説撤回。
- 信越化学が支配権を放棄: TOB仮説崩壊として再評価。

### 次点: ソニー

ソニーは、金融分離後のエンタメIP・センサー・資本配分改善を根拠に中核候補とする。現在3,280円はレポートの買い候補水準に近く、10%まで構築可能。20%まで増やすのは決算確認後でよい。

実行条件:

- 3,300円以下: 10%まで構築可能。
- 3,000円割れ: 優先買い増し。
- 3,675円接近: 新規買いは慎重化。
- KADOKAWA関連の巨額資本流出、ゲーム利益率悪化、センサー利益率低下で再評価。

### FANG+

FANG+は継続する。ただし、これは「10社すべてを同じ強度で高評価しているから買う商品」ではない。つみたて投資枠で購入可能な投資信託の中で、資本を高い成長性と再投資能力を持つ企業へ比較的集中して配分できる器として選択する。

ユーザーの基準は、産業の売上成長率ではなく、企業が受け取った資本を将来の利益、FCF、1株当たり価値へ変換できるかである。したがって、成熟企業でも高ROICでAIやクラウドへ再投資できる、自社株買いで1株当たり価値を高められる、既存プラットフォームから成長分野へ低コストで参入できるなら許容する。

実行条件:

- 当面: 月10万円継続。
- FANG+の継続判定は、成長産業への実質配分、再投資効率、指数の新陳代謝、期待収益率で行う。
- FANG+にSPCXが採用されても、それだけを理由にNASDAQ100へ自動分割しない。
- 直接SPCXと投信経由SPCXの合算比率が総金融資産の10%へ近づく場合に限り、以後の積立配分を再判定する。
- FANG+の成長産業・高効率再投資企業比率が70%を下回る場合は、NASDAQ100その他のつみたて投資枠商品への変更を検討する。
- FANG+のMU比率が高いままなら: 直接MUを優先削減。

### MU / SNDK

この2つは単純な「縮小対象」ではなく、2027年末までの需要・収益可視性を根拠に、2026年中の価格上昇を非課税で取りに行く期限付き成長投資として再分類する。恒久的な20年保有銘柄ではないが、取得価格へ戻っただけで売る対象でもない。

実行条件:

- MU特定口座3株: 戦術的な上乗せ枠。次の主要メモリ需給イベント前にT-5からT-2で売却を検討し、信越ポリマー・ソニーへの移行原資にする。
- MU NISA5株: 期限付き成長コア。2026年末を基本期限に保有し、2027年需要の織り込み進展、需給悪化、期待収益率低下で売却する。通常の四半期決算は機械的に避けない。
- SNDK NISA1株: 期限付き高ベータ成長枠。2026年末を基本期限に保有し、NAND・エンタープライズSSD需要、期待先行の急騰、需給悪化で判断する。
- 新規MU/SNDK買い増しは停止する。
- 取得価格、含み損益、直近高値は主な売却基準にしない。
- 2026年10-12月を最終出口検討期間にし、年末を待たず2027年利益が十分に織り込まれた場合は資本再配分として売却してよい。
- DRAM/NAND価格の上昇停止、在庫日数増加、ビット供給成長過多、増産前倒し、長期契約条件悪化、ハイパースケーラーCapEx減速、エンタープライズSSD需要鈍化のうち2項目で前倒し縮小する。
- 好材料に株価が反応しなくなった場合、仮説崩壊前の織り込み完了シグナルとして売却を検討する。
- MATCH Actは保有延長の単独理由ではなく、需給仮説を補強する材料として扱う。

### SPCX

SPCXは、前回の単純なTrim判定から「Strategic Option Hold」へ変更する。理由は、SpaceX/Starlinkが垂直統合された打ち上げ能力、再利用ロケットのコスト優位、衛星通信のグローバル展開、ユーザー規模拡大を同時に持つ稀有な事業モデルであり、Amazon Leo/Kuiperは資本力のある競合ではあるものの、衛星配備数、打ち上げ頻度、実運用顧客基盤でStarlinkに明確な先行優位があるためである。

ただし、これは「割安株」ではない。SPCXの課題は評価倍率、公開浮動株の少なさ、ロックアップ解除、xAI統合、Musk依存である。したがって、売却を急がず長期オプションとして保有する一方、追加購入とレバレッジは禁止する。

実行条件:

- 現在保有3株は継続保有を許容。
- 総資産比率は原則5%以内、上限でも10%以内。
- FANG+/NASDAQ100経由のSPCX間接保有が増えた場合、個別SPCXの買い増しはしない。
- StarlinkのARPUが月50ドルを下回る、Starship商業化が長期停止する、xAI損失がStarlink利益を恒常的に食う場合は再評価。
- 価格が急騰してSPCX単体が10%を超える場合のみ、一部利確を検討する。

## 5. 資金配分の実行順序

1. FANG+月10万円積立は継続。
2. MU/SNDK/SPCX/SMCIの買い増しを停止。
3. SNDKは現在価格でも売却検討可能な水準。優先的に整理候補。
4. MU特定は取得単価付近への戻りを待ち、売却候補。
5. 信越ポリマーを初期10%まで構築。
6. ソニーを10%まで構築。
7. 機動余力200万円を維持できる範囲で、信越ポリマーを15%、ソニーを15%へ増やす。
8. 信越ポリマーの強い押し目のみ、最大20%まで許容。

## 6. レバレッジ判定

レバレッジは原則として使わない。ただし使う場合は、以下をすべて満たす場合だけ許容する。

| 条件 | 判定 |
|---|---|
| 対象が信越ポリマーまたはソニー | 必須 |
| 機動余力200万円を維持 | 必須 |
| 信越ポリマー20%、ソニー20%を超えない | 必須 |
| 逆指値または撤退条件が明確 | 必須 |
| MU/SNDK/SPCX/FANG+/NASDAQ100には使わない | 必須 |

現時点では、レバレッジよりも先にMU/SNDKの整理で余力を作る方が合理的である。SPCXは売却原資ではなく、上限つきの長期オプション枠として扱う。

## 7. 最終判定

今回の再判定では、ポートフォリオの主軸を以下へ変更する。

| 旧主軸 | 新主軸 |
|---|---|
| MU/SNDKによるAIメモリ集中 | FANG+積立 + 信越ポリマー + ソニー + SPCX戦略的オプション |

明確な行動は以下。

1. **信越ポリマーをCore Buyに格上げ**し、初期10%構築を許容する。
2. **ソニーをCore Buyに格上げ**し、10%構築を許容する。
3. **FANG+は制度内の高効率成長配分枠として継続**し、SPCX採用だけでは積立分割しない。
4. **MU特定3株はイベント前の戦術売却候補**とし、T-5からT-2で決済を検討する。
5. **MU NISA5株とSNDK NISA1株は2026年末を基本期限とする期限付き成長投資**として保有し、需給悪化・評価過熱・期待収益率低下で前倒し売却する。
6. **SPCXは戦略的オプションとして継続保有**し、買い増しとレバレッジは禁止する。
7. **余力200万円は必須条件**として維持する。
