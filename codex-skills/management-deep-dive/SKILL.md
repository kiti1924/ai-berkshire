# 経営陣の深掘り：株式を通じて誰へ資本を託すか

$ARGUMENTS に対して、経営陣の質を深く調査する。

**対応する入力形式**：`会社名`または`人物名 会社名`。例：`Meituan`、`Wang Xing Meituan`、`Jensen Huang NVIDIA`

> 株式投資は、長期的には経営陣へ資本配分を委ねる行為でもある。
>
> 経営陣の評価では、平時の言葉より、監視されていない状況や困難な時期の行動を見る。

## 設計思想

通常の企業分析では、経営陣評価が経歴、持株比率、報酬だけで終わりやすい。しかし、長期returnへ強く影響するのは、誠実性、判断力、実行力、資本配分、succession、governanceである。

本Skillは`/investment-research`にある経営陣評価の深掘り版である。標準調査で経営陣評価が不確実、または経営陣自体が投資仮説の中心である場合に使う。

AIは経営陣と直接会話できないが、公開情報から次を確認できる。

- 過去の発言と実行が一致したか。
- M&A、自社株買い、配当、新規事業投資の結果。
- 困難な時期に、問題を認め、長期利益を守る判断をしたか。
- 公開された従業員、顧客、supplier、業界関係者の評価と矛盾しないか。

公開情報だけで人格を断定せず、確認済み行動、第三者評価、推定を分ける。

## 実行手順

### 手順1：実際の意思決定者を特定する

WebSearchと法定開示で次を確認する。

| 役割 | 氏名 | 在任期間 | 経歴 | 経済的持分・option | 主な権限 |
|---|---|---|---|---|---|
| CEO・取締役会議長 | | | | | |
| CFO | | | | | |
| 創業者 | | | | | |
| 実質支配者 | | | | | |
| 重要な事業責任者 | | | | | |

肩書と実権を区別する。退任した創業者が議決権、取締役会、文化、重要人事を通じて影響を持つ場合は、その根拠を示す。推測だけでshadow controllerと断定しない。

人物特定後、Taskツールで4Agentを**同一メッセージ内で並行起動**する。

1. **public-record-analyst**：株主向け書簡、earnings call、interview、social mediaなどの発言と予測。
2. **capital-allocation-analyst**：M&A、自社株買い、配当、債務、研究開発、新規事業。
3. **governance-analyst**：持株、議決権、関連当事者取引、報酬、取締役会。
4. **stakeholder-scout**：従業員、顧客、merchant、supplier、業界関係者による公開評価。

各Agentへ、調査日、対象期間、人物の正式名称、情報源優先順位、最終出力を日本語にすることを伝える。

### 手順2：能力の輪を評価する

#### 2.1 戦略判断

直近5年を基本に、経営陣の公開判断を時系列で確認する。

| 発言日 | 判断・予測 | 対象期間 | 実際の結果 | 判定 | 情報源 |
|---|---|---|---|---|---|
| | 「市場Xは…」 | | | ✅ / ⚠️ / ❌ / 判定不能 | |
| | 「今後3年は…」 | | | | |

確認する問い：

- 市場に先行した正しい判断があったか。
- 過熱時に資本規律を保ったか。
- consensusを繰り返すだけか、独立した因果理解があるか。
- 外れた予測を修正し、その理由を説明したか。

予測の正確性は、後知恵で曖昧な発言を成功扱いせず、事前に検証可能だった内容だけで評価する。

#### 2.2 実行力

| 観点 | 評価 | 証拠 |
|---|---|---|
| 戦略から実行 | 発言した計画を期限・指標どおり実行したか | |
| 組織 | 人材を採用・維持し、権限委譲できるか | |
| 危機対応 | liquidity、顧客、従業員、長期競争力を守ったか | |
| 修正速度 | 誤りを認め、resource allocationを変えたか | |
| operating discipline | 成長と費用管理を両立したか | |

### 手順3：誠実性を評価する

誠実性は最重要であるが、公開情報からの推論には限界がある。違法・不正・利益相反は一次資料で確認し、噂だけで断定しない。

#### 3.1 約束と実績

過去3年の決算call、株主向け書簡、公式interviewから具体的な約束を抽出する。

| # | 発言日 | 約束 | 期限 | 場所 | 実績 | 判定 |
|---:|---|---|---|---|---|---|
| 1 | | `2025年に事業Xを黒字化する` | | | | ✅ / ⚠️ / ❌ / 判定不能 |
| 2 | | `総額Xの自社株買いを行う` | | | | |

履行率を計算する場合、曖昧なaspirationを約束へ含めない。

| 履行率 | 参考評価 |
|---:|---|
| 80%超 | 高い一貫性を示す |
| 60～80% | 概ね実行したが、重要な差異を確認する |
| 40～60% | 約束の量、予測可能性、executionに懸念 |
| 40%未満 | 信頼性を大きく再評価する |

件数が少ない場合、percentageを過度に重視せず、各約束の重要性と難易度を説明する。

#### 3.2 困難な時期

株価下落ではなく、業績悪化、規制、製品事故、競争、liquidityなど実際の危機を選ぶ。

| 事象 | 時期 | 経営陣の初動 | 長期対応 | 後から見た結果 | 評価 |
|---|---|---|---|---|---|

確認する点：

- 問題を早期に開示したか。
- 内部要因を認めたか、外部へ責任転嫁したか。
- 短期の株価を守るため、長期価値を損なわなかったか。
- 顧客、従業員、supplier、株主の負担配分は公正だったか。

#### 3.3 stakeholderへの態度

| stakeholder | 行動 | 証拠 | 評価 |
|---|---|---|---|
| 株主 | 尊重 / 情報格差利用 / 希薄化 | | |
| 従業員 | 育成 / 公正な報酬 / 過度な負担 | | |
| 顧客 | 長期価値 / 短期収益化 | | |
| merchant・supplier | 公正な契約 / bargaining powerの濫用 | | |
| 規制当局・社会 | compliance / loophole依存 | | |

一つの匿名投稿を企業文化全体の証拠にしない。

### 手順4：資本配分を評価する

経営陣が獲得したcashをどの程度のreturnで再配分したかを確認する。

#### 4.1 主要な意思決定

**M&A**

| 日付 | 対象 | 取得額 | 通貨 | 戦略logic | integration結果 | 推定return | 評価 |
|---|---|---:|---|---|---|---:|:---:|

**自社株買い**

`tools/financial_rigor.py verify-valuation`で、取得時と現在のvaluationを検算する。

| 期間 | 総額 | 平均価格 | 当時PER・FCF Yield | 発行済株式数への効果 | 後から見た評価 |
|---|---:|---:|---:|---:|---|

買い戻し額だけでなく、SBCとのnet effectを確認する。

**配当**

| 年度 | 配当総額 | payout ratio | FCF | debt・投資需要 | 持続可能性 |
|---|---:|---:|---:|---|---|

**新規事業・研究開発**

| 開始時期 | 分野 | 累積投入 | 現在のKPI | 収益・戦略成果 | 中止基準 | 評価 |
|---|---|---:|---|---|---|:---:|

#### 4.2 scorecard

| 観点 | 評価（1～5） | 根拠 |
|---|:---:|---|
| M&A規律 | | 価格、integration、減損 |
| 自社株買い | | undervaluation、希薄化相殺、timing |
| 配当 | | FCF、debt、成長投資との整合 |
| 新規投資 | | hurdle rate、撤退規律、成果 |
| cash管理 | | liquidity、過剰cash、debt maturity |
| **総合** | | |

好機があるときは投資し、好機がないときは低価格の自社株買い、配当、債務削減などから最良の用途を選んだかを評価する。高値のM&Aを「成長投資」という言葉だけで正当化しない。

### 手順5：governanceを評価する

#### 5.1 所有・議決権

| 項目 | 内容 | risk |
|---|---|---|
| dual-class・super-voting | | |
| 創業者・支配株主の経済的持分 | | |
| 議決権比率 | | |
| VIEその他の契約構造 | | |
| 独立取締役の独立性・tenure | | |
| 主要株主・役員の売買 | | |

経済的持分と議決権を分け、基準日を付ける。

#### 5.2 報酬

| 役員 | 総報酬 | 通貨・年度 | 純利益・FCF比率 | peer比較 | 長期整合性 |
|---|---:|---|---:|---|---|

base salary、bonus、SBC、option、公正価値を区別する。報酬設計が売上高、調整後利益、株価など一つの短期指標へ偏っていないか確認する。

#### 5.3 関連当事者取引

| 相手方 | 関係 | 内容 | 金額 | 条件の公正性 | risk |
|---|---|---|---:|---|---|

法定開示、audit committeeの承認、pricing、継続性を確認する。

### 手順8：報告を作る

```text
1. 結論
2. 重要人物と意思決定構造
3. 誠実性
   - 約束の履行
   - 困難な時期
   - stakeholderへの態度
4. 能力
   - 戦略判断
   - execution
   - 資本配分
5. governance
   - 所有・議決権
   - 報酬
   - 関連当事者取引
6. 側面情報
7. successionとkey person risk
8. 総合評価、反対意見、未確認事項
9. 出典
```

#### 総合評価

| 観点 | 参考weight | 評価（1～5） | 根拠 |
|---|:---:|:---:|---|
| 誠実性 | 35% | | |
| 戦略・execution | 25% | | |
| 資本配分 | 25% | | |
| governance | 15% | | |
| **総合** | 100% | | |

weightによる精密値を過信せず、誠実性に重大な問題があれば総合scoreにかかわらず投資仮説を再評価する。

#### 「誰へ託すか」の3問

1. **誠実性を支持する行動があるか。**
2. **戦略、execution、資本配分の能力を確認できるか。**
3. **10年間資本を託す前提で、組織とgovernanceを含めて信頼できるか。**

3問すべてが肯定なら高評価、能力は高いが誠実性に疑義がある場合は不適格、情報不足なら判定保留とする。

## 重要原則

- **誠実性は重大な否決条件になり得る**。ただし噂だけで断定しない。
- **言葉より行動を見る**。発言と実績を同じ期間・定義で比較する。
- **困難な時期を見る**。順風時の結果だけで能力を判断しない。
- **資本配分を検算する**。取得価格、return、希薄化、機会費用を確認する。
- **経営陣へ感情移入しない**。好意的な人物にも反対証拠を適用する。
- **事実、第三者評価、推定、分析者の評価を分ける。**
- **本資料は調査・学習用であり、投資助言ではない。**
