# ポートフォリオ・レビュー：企業調査から資産配分へ

$ARGUMENTS に対して、投資ポートフォリオの点検と改善案の作成を行う。

**対応する入力形式**：

- 比率による保有一覧：`Tencent 30%, Meituan 20%, Kweichow Moutai 20%, NVIDIA 15%, 現金15%`
- 数量と取得価格：`Tencent 500株 @480 HKD, Meituan 1,000株 @130 HKD, ...`
- `私のポートフォリオ`：保存済みの`reports/portfolio-latest.md`がある場合に読み込む。

> 「分散投資は無知に対する防御である。自分が何をしているか理解しているなら、広すぎる分散は必ずしも必要ではない。」— Buffettの趣旨
>
> 「本当に優れた投資機会は、一生のうち両手で数えられるほどしかない。」— 李録の投資姿勢を要約した表現

## 設計思想

企業を調べることは投資の半分にすぎない。残りは**ポートフォリオ全体の意思決定**である。

- どれだけ持つか：position sizing。
- どの資金を使うか：新規資金か、既存資産からの入替えか。
- 既存保有と同じリスクへ偏っていないか：相関と共通要因。
- 最良の資産配分は何か：機会費用。

個別銘柄を孤立して評価せず、「これは現在選べる選択肢の中で最も良い資金用途か」を問う。

## 実行手順

### 手順1：保有状況を整理する

入力から現在の保有資産を抽出し、次の形式へ正規化する。

| 対象 | ticker | 数量 | 平均取得単価 | 現在値 | 時価 | 比率 | 評価損益 |
|---|---|---:|---:|---:|---:|---:|---:|

- 入力が比率だけなら、金額を推測せず比率ベースで分析する。
- 通貨が複数ある場合は、各通貨と換算基準日を明示する。
- 現金、債券、投資信託、未上場資産なども、ユーザーが示した範囲で含める。
- `reports/portfolio-latest.md`があれば読み、今回の入力と矛盾する項目を確認してから更新する。

### 手順2：最新データを取得する

Taskツールで保有対象ごとにバックグラウンドAgentを起動し、WebSearchと一次資料から並行して次を取得する。

1. 現在価格と評価指標（PER、PBR、配当利回りなど）。
2. 直近決算での重要な財務変化。
3. 前回確認後の重要事象。
4. 市場予想（予想PER、利益予想、目標価格など）。市場予想は事実や独自推定と分ける。

各保有対象の評価データは次で検証する。

```bash
python3 tools/financial_rigor.py verify-valuation ...
```

情報量をA/B/Cで評価し、Cの対象では結論を低確度と明記する。株価・為替・時価総額には具体的な基準日時を付ける。

### 手順3：各positionを点検する

| 対象 | 現在PER | 購入仮説の変化 | 仮説健全性 | 比率の評価 |
|---|:---:|:---:|:---:|---|
| Tencent | 18倍 | 変化なし | 8/10 | 妥当 |
| Meituan | 25倍 | 競争激化 | 6/10 | やや高い。縮小を検討 |

各positionについて次に答える。

- [ ] 今日まったく保有していなくても、現在価格で新たに買うか。
- [ ] 明日から5年間売買できなくても、安心して保有できるか。
- [ ] 購入時の投資仮説は維持されているか。
- [ ] 現在の比率は、確信度だけでなく下方リスクと流動性に見合うか。

「10年間持ちたくない株式は、短期間でも安易に持たない」という長期保有の考え方を判断軸として使う。

### 手順4：ポートフォリオ全体を分析する

#### 4.1 集中度

| 指標 | 現在値 | 参考範囲 | 判定 |
|---|---:|---:|---|
| 最大position比率 | | < 40% | |
| 上位3positionの合計 | | 50～80% | |
| 保有対象数 | | 5～15 | |
| 現金比率 | | 10～30%を参考とし、資金用途とリスク許容度で調整 | |

この範囲は機械的な合否基準ではない。3～5社へ集中する方法は、各社を深く理解し、損失許容度と流動性を確保できる場合にのみ成立する。逆に、銘柄数が多くても同じ要因へ偏っていれば分散とはいえない。

#### 4.2 相関と共通リスク

表面的な業種分類だけでなく、収益源、地域、通貨、規制、金利、商品価格、供給網などの共通要因を特定する。

| 保有A | 保有B | 共通要因 | 想定リスク |
|---|---|---|---|
| Tencent | Kuaishou | 中国インターネット・広告 | 規制や広告需要の同時悪化 |
| NVIDIA | TSMC | AI計算資源の供給網 | AI設備投資と半導体cycleの同時変動 |
| Meituan | PDD | 中国個人消費 | 消費減速と価格競争の同時進行 |

**確認項目**：

- [ ] 50%超が一つのテーマまたは業界へ依存していないか。
- [ ] 50%超が一つの国、通貨、規制環境へ依存していないか。
- [ ] 米中関係の悪化で、どの保有対象が同時に影響を受けるか。
- [ ] 世界景気後退で、利益と評価倍率が同時に悪化した場合の影響はどの程度か。
- [ ] 一つの供給者、顧客、金利水準、商品価格に隠れた集中がないか。

#### 4.3 機会費用

**各単位の資金を、リスク調整後の期待価値が最も高い用途へ配分できているかを確認する。**

全保有対象を期待年率リターンで並べる。

| 順位 | 対象 | 現在比率 | 期待年率リターン | 確信度 | リスク調整後の評価 |
|:---:|---|---:|---:|:---:|---:|
| 1 | | | | | |
| 2 | | | | | |
| ... | | | | | |

期待リターンは次を参考に、楽観・基準・悲観の範囲で計算する。

```bash
python3 tools/financial_rigor.py three-scenario ...
```

- **基本的な概算**：FCF Yield + 持続可能な成長率。
- **value型の確認**：評価倍率の正常化 + 利益成長 + 配当。
- **growth型の確認**：利益成長と妥当な評価倍率の変化。

成長率を二重計上しない。確信度は、情報量、事業の予測可能性、財務耐久性、評価幅を考慮して説明する。

**重要な問い**：最下位の保有対象は、税金、取引費用、リスクを考慮しても現金または低リスク資産より魅力的か。魅力がない場合は、保有継続の根拠を再確認する。

#### 4.4 ストレステスト

| シナリオ | 仮定 | ポートフォリオへの推定影響 | 推定最大drawdown |
|---|---|---|---:|
| 世界景気後退 | 企業利益が20～30%減少 | | |
| 米中対立の激化 | 中国関連株のrisk premiumが大幅上昇 | | |
| 金利上昇 | 10年国債利回りが6%へ上昇 | | |
| テクノロジー評価の収縮 | テクノロジー株のPERが40%低下 | | |

各シナリオについて、定性的評価と概算レンジを示す。

- 最も影響を受ける保有対象は何か。
- 利益減少と評価倍率低下を区別し、両方が起きる場合を確認する。
- ポートフォリオ全体で許容できる損失範囲か。
- 現金、資産配分、position縮小などの対策が必要か。

根拠のない精密値を示さず、入力値と仮定を明記する。

### 手順5：改善案を作る

#### 5.1 配分変更案

| 対応 | 対象 | 現在比率 | 参考比率 | 根拠 |
|---|---|---:|---:|---|
| 増加を検討 | | | | |
| 縮小を検討 | | | | |
| 撤退を検討 | | | | |
| 新規候補 | | | | |
| 維持 | | | | |

対応案は、投資仮説、評価、相関、流動性、税金、取引費用、ユーザーの制約を根拠にする。保有数量が不明な場合は、具体的な注文数量を作らない。

#### 5.2 代替候補の探し方

現金より魅力が低いpositionがある、または現金比率が高く新規候補を探す必要がある場合は、`/industry-research`または`/investment-checklist`で体系的に調査する。本Skill内で根拠なく個別銘柄を推薦しない。

#### 5.3 現金管理

| 現在の現金比率 | 参考とする現金比率 | 根拠 |
|:---:|:---:|---|

現金は「何もしていない状態」ではなく、損失耐性、生活資金、将来の機会を支えるpositionである。優れた機会がないときに、無理に投資比率を上げない。

### 手順6：ポートフォリオ報告を出力する

#### 報告構成

```text
1. 結論
2. 前提、データ基準日、通貨換算
3. ポートフォリオ概要（保有表と構成比の説明）
4. 各positionの点検
5. ポートフォリオ分析
   - 集中度
   - 相関と共通リスク
   - 機会費用
   - ストレステスト
6. 配分変更の選択肢と根拠
7. 次回確認時期と重点項目
8. 出典
```

#### 結論で必ず答えること

1. **全体の健全性**：良好 / おおむね良好 / 要調整 / 重大な問題あり。
2. **最も優先すべき一つの対応**：維持、追加調査、特定positionの縮小検討などを、理由とともに示す。
3. **現在最大のリスク**：銘柄固有ではなく、ポートフォリオ全体への影響も示す。
4. **最も不確実な前提**：追加データが必要な点を明記する。

### 手順7：ポートフォリオファイルを保存する

`reports/portfolio-latest.md`へ次を保存する。

- 最新の保有表。
- 確認日、価格・為替の基準日時、結論。
- 配分変更履歴。過去記録を削除せず追記する。
- 次回確認時期と重点項目。
- 主要な出典と未確認事項。

## 重要原則

- **すべての資金には機会費用がある**：平均的な資産を持つことは、より優れた機会を逃すことでもある。
- **集中それ自体より、理解不足が危険である**：ただし、理解しているという主観だけで過大positionを正当化しない。
- **現金もpositionである**：優れた機会がないときに待つ余地を残す。
- **ポートフォリオ全体を優先する**：優れた銘柄でも比率が過大なら、全体の損失を大きくし得る。
- **定期的に確認し、過剰売買を避ける**：原則として四半期ごと、または投資仮説に関わる重大事象の発生時に確認する。
- **事実、推定、仮定、評価を分ける**：市場予想やscenario計算を確定事実として扱わない。
- **本資料は調査・学習用であり、投資助言ではない。**
