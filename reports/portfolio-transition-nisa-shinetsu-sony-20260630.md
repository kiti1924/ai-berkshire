# 積立NISAを維持しつつ信越ポリマー・ソニー重点へ移行する計画

**作成日**: 2026-06-30

**目的**: FANG+積立を維持しながら、直接メモリ株偏重を落とし、信越ポリマーとソニーを将来ポートフォリオの重点枠へ移行する。

## 1. 中核方針

月10万円のつみたて投資枠は、長期成長枠として維持する。FANG+は「10社すべてを同じ強度で高評価しているから買う商品」ではない。つみたて投資枠で購入可能な投資信託の中で、資本を高い成長性と再投資能力を持つ企業へ比較的集中して配分できる手段として選択する。

判断基準は、売上成長率そのものではなく、企業が受け取った資本を将来の利益、FCF、1株当たり価値へ効率よく変換できるかである。成熟企業であっても、既存事業から生じる余剰資金をAI、クラウド、半導体、データ基盤、その他の成長分野へ高い収益率で再投資できる限り、FANG+内での保有を許容する。

一方で、FANG+は10銘柄集中型であり、MUや将来のSPCX採用により、個別株側の保有とリスクが重複する。MU/SNDKは直接メモリ株として削減対象にするが、SPCXはSpaceX/Starlinkの独自性を反映し、売却原資ではなく上限つきの戦略的オプション枠として扱う。

優先順位は以下とする。

1. 機動用現金・MMFを200万円以上に維持する。
2. MU/SNDK/SPCXの個別買い増しを停止する。
3. MU特定はイベント前の戦術売却候補、MU/SNDKのNISA分は2026年末を基本期限とする期限付き成長投資にする。
4. 信越ポリマーを最大20%までの重点候補にする。
5. ソニーを10-20%の中核候補にする。
6. SPCXは5%目安、最大10%以内の長期オプション枠として保有する。
7. 任天堂と上流半導体は第2段階の候補にする。

## 3. 積立NISAルール

### 3.1 FANG+を継続する条件

以下を満たす限り、FANG+月10万円を継続する。

- FANG+が、つみたて投資枠内で成長企業への高濃度配分手段として最も有効である。
- 直接成長産業と、高効率で成長分野へ再投資する成熟企業の合計が、おおむね70-80%以上である。
- 構成企業のROIC、FCF成長、株式報酬控除後の1株当たりFCF、自社株買いによる実株数減少が劣化していない。
- SPCXが採用された場合でも、直接SPCXと投信経由SPCXの実質比率が総金融資産の10%へ近づいていない。

### 3.2 NASDAQ100へ一部変更する条件

SPCXのFANG+採用だけを理由に、NASDAQ100へ自動分割しない。NASDAQ100へ一部変更するのは、以下のいずれかが起きた場合に限る。

- 直接SPCXとFANG+/NASDAQ100経由SPCXの合算比率が総金融資産の10%へ近づく。
- FANG+の成長産業・高効率再投資企業比率が70%を下回る。
- FANG+構成銘柄の再投資効率が悪化し、株式報酬控除後の1株当たりFCF成長が鈍化する。
- 指数の新陳代謝が弱まり、過去の大型勝者に固定された状態になる。

変更する場合の一時的な配分は以下とする。

| 積立先 | 月額 |
|---|---:|
| iFreeNEXT FANG+インデックス | 5万円 |
| iFreeNEXT NASDAQ100インデックス | 5万円 |

NASDAQ100はFANG+より分散されるが、資本配分の意図は薄まりやすい。分散改善だけを理由に移すのではなく、FANG+が成長投資手段として有効でなくなった場合のサブ候補として扱う。

### 3.3 FANG+の四半期レビュー

FANG+は、四半期または半年ごとに以下で評価する。

| 評価項目 | 継続基準 |
|---|---|
| 成長産業への実質配分 | 直接成長産業 + 高効率再投資企業が70-80%以上 |
| 再投資効率 | ROIC、FCF成長、株式報酬控除後の1株当たりFCFが健全 |
| 指数の新陳代謝 | SPCXのような新しい大規模成長企業を適時取り込める |
| 期待収益率 | 1株当たり利益成長率 + 株主還元率 +/- 評価倍率変化が十分 |

### 3.4 個別株との重複管理

FANG+を継続する代わりに、以下の個別株買い増しは禁止する。

- MU
- SNDK
- SPCX
- SMCI

MUとSPCXは、FANG+/NASDAQ100経由での間接保有を含めて実質比率を確認する。MUが過大な場合は個別MUを先に削減する。SPCXは過大な場合でも原則は新規買い増し停止で対応し、売却は単体比率が10%を超えた場合に限って検討する。

## 4. 個別株の実行ルール

### 4.1 米国メモリ株の出口ルール

MU/SNDKは単純な「縮小対象」ではなく、2027年末までの需要・収益可視性を根拠に、2026年中の価格上昇を非課税で取りに行く期限付き成長投資として扱う。永久保有の中核ではないが、取得価格へ戻っただけで売る対象でもない。

この前提では、口座ごとに役割を分ける。

| 保有 | 位置づけ | 基本方針 |
|---|---|---|
| MU特定口座3株 | 戦術的な上乗せ枠 | メモリ需給イベント前に分割売却を検討し、ソニー・信越ポリマーへの移行原資にする |
| MU NISA5株 | 期限付き成長コア | 2026年末を基本期限に保有し、2027年需要の織り込み進展または需給悪化で売却する |
| SNDK NISA1株 | 期限付き高ベータ成長枠 | 2026年末を基本期限に保有し、NAND・エンタープライズSSD需要、期待過熱、需給悪化で判断する |
| SPCX NISA | 戦略的オプション | SpaceX/Starlinkの独自性を長期オプションとして保持する |

イベント回避ルールは、主としてMU特定口座3株に適用する。次のうち最も早い公式イベントを基準日Tにし、T-7営業日に買い注文を取消し、T-5からT-2までにMU特定口座を売却する。

- Samsungの第2四半期速報または本決算
- SK hynixの第2四半期決算
- SanDiskの次回決算
- Micronの次回決算
- 大手各社による価格改定、増産、設備投資、在庫見通しの公式発表

NISA分は通常の四半期決算を機械的に避けない。NISAは非課税の上昇余地を取りに行く枠であり、毎回の決算前に閉じると上昇の主要部分を取り逃がす可能性がある。NISA分をイベント前に売るのは、需給仮説が悪化した場合、期待が過熱して2027年利益を十分以上に織り込んだ場合、またはポートフォリオ比率が過大になった場合に限る。

売却基準:

- 時間基準: 2026年10-12月を最終的な出口検討期間にする。2027年の好業績が十分に織り込まれたと判断できれば、年末を待たず売却してよい。
- ファンダメンタル基準: DRAM/NAND価格の上昇停止、在庫日数増加、ビット供給成長が需要成長を上回る、Samsung/SK hynix/Micronの増産前倒し、長期契約条件の悪化、ハイパースケーラーCapEx減速、エンタープライズSSD需要鈍化のうち2項目で前倒し縮小する。
- 評価基準: 2027年利益を十分以上に織り込んだ、期待収益率がソニーや信越ポリマーを下回った、MU/SNDK比率が過大になった、好材料に株価が反応しなくなった場合は、投資成功後の資本再配分として売却する。

取得価格、含み損益、直近高値は売却判断の主基準にしない。新規MU/SNDK買い増しは停止し、2026年後半から年末にかけて、メモリ株から信越ポリマー・ソニーなど恒久的中核候補へ資本を移す。

### 4.2 信越ポリマー

信越ポリマーは、最大20%まで許容する重点枠とする。ただし小型株、親子上場、流動性リスクがあるため一括で20%にしない。

| 価格・条件 | 行動 |
|---|---|
| 2,400円台 | 初期10%まで許容 |
| 2,000円前後以下 | 15-20%まで段階買いを検討 |
| 3,200円以上 | 新規買い停止、一部利益確定を検討 |
| 精密成形品利益率10%以下 | 仮説撤回・売却検討 |
| 信越化学が支配権を放棄 | TOB仮説崩壊として売却検討 |

根拠は、同社レポートにおける以下の評価に置く。

- FY2026売上高1,151億円、営業利益140億円、親会社当期純利益99億円。
- 自己資本比率84.4%、広義ネットキャッシュ474億円。
- 精密成形品が利益構成比72.8%。
- 基準シナリオの10年後目標株価3,969.8円、悲観シナリオ1,477.8円。

### 4.3 ソニー

ソニーは、信越ポリマーの初期ポジション構築後に積み上げる中核候補とする。

| 価格・条件 | 行動 |
|---|---|
| 3,268円近辺 | 買い候補 |
| 3,000円割れ | 優先的に積み増し検討 |
| 3年基準シナリオ3,675円接近 | 新規買いは慎重化 |
| KADOKAWA関連の巨額資本流出 | 再評価 |
| ゲーム利益率の悪化 | 再評価 |
| センサー利益率の低下 | 再評価 |

根拠は、ソニーレポートにおける以下の評価に置く。

- 金融スピンオフ後の「ソニー2.0」への純化。
- 継続事業FCF約1.07兆円、FCFコンバージョン率103.3%。
- 実質EV/EBITDA 6.45倍、FCF利回り5.51%。
- 音楽、アニメIP、積層型CMOSセンサーの複合モート。
- 3年後基準シナリオ株価3,675.4円、楽観シナリオ5,298.9円。

## 5. レバレッジ・リスク管理

レバレッジは、明確な割安株に限定する。対象は原則として、信越ポリマーとソニーのように一次資料ベースで安全域が確認済みの銘柄に限る。

以下へのレバレッジは禁止する。

- MU
- SNDK
- SPCX
- SMCI
- FANG+
- NASDAQ100

レバレッジ込みでも、機動用現金・MMF200万円を割らない。200万円を割る場合は、新規買付よりもポジション縮小を優先する。

## 6. 月次レビュー項目

毎月末に以下を確認する。

| 項目 | 判定基準 |
|---|---|
| 機動余力 | 200万円以上を維持 |
| 信越ポリマー比率 | 20%以下 |
| ソニー比率 | 20%以下 |
| FANG+の役割 | 制度内の高効率な成長資本配分手段として機能しているか |
| FANG+の成長産業比率 | 直接成長産業 + 高効率再投資企業が70-80%以上か |
| FANG+の再投資効率 | ROIC、1株当たりFCF、希薄化、自社株買いを確認 |
| FANG+の新陳代謝 | SPCXのような新しい大規模成長企業を取り込めるか |
| 直接MU + FANG+経由MU | 過大なら直接MUを削減 |
| 直接SPCX + 投信経由SPCX | 合算比率が総金融資産の10%へ近づくか |
| メモリ株 | 新規買い増しなし |

## 7. 決算レビュー項目

### 信越ポリマー

- 精密成形品部門の利益率。
- 半導体容器需要。
- 自己資本比率とネットキャッシュ。
- 親会社である信越化学の保有方針。
- 中期経営計画に対する進捗。

### ソニー

- 継続事業FCF。
- 自社株買いと株主還元。
- ゲーム事業の利益率。
- 音楽・アニメ事業の成長率。
- イメージセンサーの利益率とApple依存。
- KADOKAWA関連の追加資本流出。

### MU/SNDK

- MU特定口座3株について、次の主要メモリ需給イベント日程が公表されたか。
- MU特定口座をT-5からT-2までに売却する必要があるか。
- MU NISA5株とSNDK NISA1株について、2026年10-12月の出口検討期間に入ったか。
- DRAM/NAND価格の上昇停止、在庫日数増加、供給成長過多、増産前倒し、長期契約条件悪化、ハイパースケーラーCapEx減速、エンタープライズSSD需要鈍化のうち2項目が出たか。
- 2027年利益の織り込みが進み、期待収益率がソニーや信越ポリマーを下回っていないか。
- MU/SNDKの総資産比率が過大になっていないか。
- 好材料に株価が反応しなくなっていないか。

## 8. 実行順序

1. FANG+月10万円積立は継続する。
2. MU/SNDK/SPCX/SMCIの個別買い増しを停止する。
3. MU特定口座3株は、次の主要メモリ需給イベントが公表されたら基準日Tを置き、T-5からT-2で売却する。
4. MU NISA5株は、2026年末を基本期限とする期限付き成長コアとして保有し、需給悪化・評価過熱・期待収益率低下で前倒し売却する。
5. SNDK NISA1株は、2026年末を基本期限とする高ベータ成長枠として保有し、NAND固有の悪化または期待先行の急騰時に売却する。
6. 2026年10-12月に、MU/SNDK NISA分を恒久保有へ延長するか、信越ポリマー・ソニー・現金へ移すかを最終再判定する。
7. 信越ポリマーを10%まで構築する。
8. 信越ポリマーが2,000円前後以下へ下がる場合、15-20%まで段階的に増やす。
9. ソニーを10%まで構築し、価格条件と決算確認で20%まで増やす。
10. SPCXは継続保有し、直接保有と投信経由の合算比率が10%へ近づくまでは売却しない。
11. FANG+はSPCX採用だけでは分割せず、成長産業比率・再投資効率・実質SPCX比率のいずれかが基準を外れた時だけ積立先を再判定する。
12. 任天堂と上流半導体は、上記が整った後の第2段階として検討する。

## 9. 注意事項

本計画は学習・研究目的のポートフォリオ運用メモであり、投資助言ではない。実際の売買は、証券口座上の税制、NISA枠、為替、信用余力、手数料、生活資金を確認したうえで自己責任で行う。
