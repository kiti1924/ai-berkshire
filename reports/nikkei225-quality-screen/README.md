# 品質スクリーニング結果：日経225

**スクリーニング日**: 2026-06-30
**データ基準日**: 2026-03-31（2025年度決算ベース）
**対象企業数**: 225社（2026年4月1日入替反映済）
**出典**: macrotrends.net, stockanalysis.com, EDINET, 日本経済新聞, 有価証券報告書

---

## 結論

日経225構成銘柄225社に対して7指標の品質スクリーニングを実施した結果、**約80社が「通過」、約90社が「除外」、残りは「境界・データ不足」**と判定された。

**最重要ファインディング**:
1. 日本企業全体のROE中央値は上昇傾向（2026年6月時点で日経平均構成銘柄のROE加重平均は約10.6%）だが、個別企業ではばらつきが大きい
2. 通過率が高いセクター：**情報通信・サービス・精密機器・半導体関連**（60-80%通過）
3. 通過率が低いセクター：**銀行・鉄鋼・素材・海運・電力・建設**（20-30%通過）
4. 銀行は資本規制の性質上ROEが構造的に低い（業種平均約5.5%）が、例外扱いにはせず、本スクリーニングでは銀行の大半がROE基準で除外
5. **商社**は売上総利益率が構造的に低いが、高いROE（10-15%）とOCF/NIによりException C（高回転・薄利型モデル）で通過できるケースが多い

## セクター要約

**通過率**: ~80社 / 225社 ≈ 36%（一次データが取得できた企業ベースでは約50%）

**業界全体の品質評価**: 日本の代表的上場企業225社のうち、バフェット／段永平流の「一流企業」基準を満たすのは約3分の1。これは日本企業全体のROE改善傾向（加重平均10.6%）の中でも、個別企業のばらつきが大きいことを反映している。

| 品質区分 | 該当企業 | 共通点 |
|---|---|---|
| **一流**（全指標通過かつ高ROE>15%） | キーエンス, 東京エレクトロン, アドバンテスト, ディスコ, SCREEN HD, HOYA, 伊藤忠商事, 東京海上HD, ファーストリテイリング, リクルートHD, 任天堂 | 無形資産・技術優位・強的価格決定力 |
| **合格**（全指標通過だが平均的） | トヨタ, SMC, ダイキン, NTT, KDDI, ソフトバンク, キッコーマン, 味の素, 鹿島建設, 三菱UFJ, JR東海, ブリヂストン, 日立 | 安定した収益基盤・優良だが突出した競争優位は中程度 |
| **除外** | パナソニック, 京セラ, 日産, 日本製鉄, 本田技研, JR東日本, SOMPO, 東京電力, 武田薬品 | ROE基準未達または純利益率が構造的に低い／循環的 |

**業界選定の結論**: 日経225からさらに深掘りする価値が最も高いのは以下の2～3セクターである。
1. **半導体・精密機器セクター**（キーエンス, 東京エレクトロン, ディスコ, アドバンテスト, HOYA, SCREEN HD）— 全指標を優に超え、グローバル競争優位が明確
2. **情報通信・ITサービス**（KDDI, リクルートHD, ソフトバンク, LINEヤフー）— 高いROEとキャッシュ創出力
3. **商社（総合商社）** — 伊藤忠商事・三菱商事等、低粗利率という見かけ上の弱点を例外規定で補正した上でも、ROEとFCF創出力が際立つ

## 境界・不確実な判定

- **武田薬品(4502)**：買収負債の影響でROEが6-8%と基準未達。2024-25年度は改善傾向だが10年平均では未達。例外A（戦略投資期）の可能性があるが、上場後10年以上経過のため適用不可。
- **キリンホールディングス(2503)**：ROE 6-8%で微妙。例外Aは上場後10年超のため不可。ファーマ事業の成長次第。
- **エーザイ(4523)**：アルツハイマー薬レカネマブの承認状況次第。現在ROE 5-8%で境界線上。
- **デンソー(6902)**：粗利益率14-15.5%で閾値(15%)の境界。トヨタグループ内の構造改革次第。
- **いすゞ自動車(7202)**：ROE 10%超で通過だが、OCF/NIが変動大きく境界。
- **荏原製作所(6361)**：TTMベースのFCFが設備投資サイクルでマイナスだが長期ではプラス。設備投資期とみなせる。
- **海運3社（郵船・商船三井・川崎汽船）**：ROE・FCFは好況期には基準を超えるが、市況サイクルが激しく長期平均では未達の可能性。景気循環型業界のため、好不況を含む完全なサイクル評価が必要。
