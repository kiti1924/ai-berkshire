**使用スキル**: `portfolio-review`  
**目的**: FANG+積立を維持しながら、MU/SNDKの直接メモリ偏重を減らし、SPCXは戦略的オプション枠として管理し、信越ポリマーとソニーを重点化する判断を明確な基準で再判定する。

## 4. 結論

### 最優先: 信越ポリマー

信越ポリマーは、現在価格2,400円台で初期10%まで許容できる。最大20%まで認めるが、これは一括ではなく段階的に行う。

実行条件:

- 2,400円台: 初期10%まで構築。
- 2,000円前後以下: 15-20%まで段階買い。
- 3,200円以上: 新規買い停止、一部利確検討。
- 精密成形品利益率10%以下: 仮説撤回。
- 信越化学が支配権を放棄: TOB仮説崩壊として再評価。

### 次点: ソニー

ソニーは、金融分離後のエンタメIP・センサー・資本配分改善を根拠に中核候補とする。現在3,280円はレポートの買い候補水準に近く、10%まで構築可能。20%まで増やすのは決算確認後でよい。

実行条件:

- 3,300円以下: 10%まで構築可能。
- 3,000円割れ: 優先買い増し。
- 3,675円接近: 新規買いは慎重化。
- KADOKAWA関連の巨額資本流出、ゲーム利益率悪化、センサー利益率低下で再評価。

### FANG+

FANG+は継続する。ただし、これは「10社すべてを同じ強度で高評価しているから買う商品」ではない。つみたて投資枠で購入可能な投資信託の中で、資本を高い成長性と再投資能力を持つ企業へ比較的集中して配分できる器として選択する。

ユーザーの基準は、産業の売上成長率ではなく、企業が受け取った資本を将来の利益、FCF、1株当たり価値へ変換できるかである。したがって、成熟企業でも高ROICでAIやクラウドへ再投資できる、自社株買いで1株当たり価値を高められる、既存プラットフォームから成長分野へ低コストで参入できるなら許容する。

実行条件:

- 当面: 月10万円継続。
- FANG+の継続判定は、成長産業への実質配分、再投資効率、指数の新陳代謝、期待収益率で行う。
- FANG+にSPCXが採用されても、それだけを理由にNASDAQ100へ自動分割しない。
- SPCX実質合算比率が8%に達した場合に、以後の積立配分を再判定する。
- FANG+の成長産業・高効率再投資企業比率が70%を下回る場合は、NASDAQ100その他のつみたて投資枠商品への変更を検討する。
- FANG+のMU比率が高いままなら: 直接MUを優先削減。

### MU / SNDK

この2つは単純な「縮小対象」ではなく、2027年末までの需要・収益可視性を根拠に、2026年中の価格上昇を非課税で取りに行く期限付き成長投資として再分類する。恒久的な20年保有銘柄ではないが、取得価格へ戻っただけで売る対象でもない。

実行条件:

- MU特定口座3株: 戦術的な上乗せ枠。Micron、Samsung、SK hynixのうち最も早い予定イベントまで資金集中を取り、T-5から決済開始、T-3原則完了、T-2最終期限で売却する。取得単価への回復は条件にしない。
- MU NISA5株: 期限付き成長コア。2026年末を基本期限に保有し、2027年需要の織り込み進展、需給悪化、期待収益率低下で売却する。通常の四半期決算は機械的に避けない。
- SNDK NISA1株: 期限付き高ベータ成長枠。2026年末を基本期限に保有し、NAND・エンタープライズSSD需要、期待先行の急騰、需給悪化で判断する。SNDKやキオクシアの決算はNAND確認イベントであり、MU特定口座の機械的決済基準にはしない。
- 突発イベントは即時再判定する。対象は契約価格引き下げ、増産・稼働率引き上げ、顧客契約の取消し、在庫急増、輸出規制や政策の重大変更である。
- 新規MU/SNDK買い増しは停止する。
- 取得価格、含み損益、直近高値は主な売却基準にしない。
- 2026年10-12月を最終出口検討期間にし、原則として2026年末までに出口を決める。延長する場合も恒久保有ではなく、次の決算または3-6か月後までの新たな期限付き投資として再設定する。
- 延長条件は、2027年需要が契約・前払い・ガイダンスでさらに強化されたこと、供給増加が需要増加を上回る兆候がないこと、NISA非課税効果を含む期待収益率がソニー・信越ポリマーより依然高いことの3条件を満たす場合に限定する。
- ハードシグナルは1件で前倒し売却する。契約価格またはASPの明確な下落、在庫急増、ビット供給成長が需要成長を明確に上回るガイダンス、長期契約の取消し・下方再交渉、業績ガイダンス引き下げが該当する。
- ソフトシグナルは2件で前倒し売却する。価格上昇率の鈍化・横ばい化、リードタイム短縮、設備投資加速、好材料に対する株価反応低下、SOXに対する相対的弱さ、顧客の在庫調整コメントが該当する。
- 好材料に反応しないことは特定口座の早期決済には使えるが、NISA売却にはもう1つのシグナルを必要とする。
- MATCH Actは保有延長の単独理由ではなく、需給仮説を補強する材料として扱う。

### SPCX

SPCXは、前回の単純なTrim判定から「Strategic Option Hold」へ変更する。Amazon等の競合はあるが、SpaceX/Starlinkは打ち上げ頻度、再利用ロケット、垂直統合、通信サービスまで含めた独自のコスト・品質改善モデルを持つ。売却原資ではなく、上限付きの長期オプションとして管理する。

SPCXは単体比率ではなく、常に実質合算比率で管理する。

`SPCX実質比率 = (直接SPCX時価 + FANG+残高 x FANG+内SPCX比率 + NASDAQ100残高 x NASDAQ100内SPCX比率) / 総金融資産`

実行条件:

- 実質比率8%未満: 継続保有。
- 実質比率8-10%: 新規の間接配分を含めて再確認。
- 実質比率10%超: 直接SPCXの一部利確を検討。
- 事業仮説崩壊: 比率に関係なく売却再判定。
- 買い増しとレバレッジは禁止。

## 5. 資金配分の実行順序

1. 未拘束の現金・MMF200万円を先に確保する。信用取引の担保に算入した資金は機動余力に含めない。
2. FANG+月10万円積立は制度内の高効率成長配分枠として継続する。
3. MU特定口座3株は、次の対象イベントまで資金集中を取る戦術枠として保有し、T-5から決済開始、T-3原則完了、T-2最終期限で売却する。取得単価への回復は条件にしない。
4. MU NISA5株およびSNDK NISA1株は2026年10-12月の最終レビューまで原則保有し、需給悪化・評価過熱・期待収益率低下が確認された場合のみ前倒し売却する。
5. 信越ポリマーとソニーは固定順序で買わず、各時点の安全域、期待収益率、資金余力を比較し、条件を先に満たした方から段階取得する。
6. SPCXは実質合算比率8%で注意、10%超で直接SPCXの一部利確を検討する。

## 6. レバレッジ判定

移行完了までは、信越ポリマー・ソニーを含めて新規レバレッジを停止する。現在はリスクオフ方針と未拘束の現金・MMF200万円を優先する。

将来レバレッジを再開する場合の必須条件:

| 条件 | 判定 |
|---|---|
| 対象が信越ポリマーまたはソニー | 必須 |
| 未拘束の現金・MMF200万円を維持 | 必須 |
| 信越ポリマー20%、ソニー20%を超えない | 必須 |
| 逆指値または撤退条件が明確 | 必須 |
| MU/SNDK/SPCX/FANG+/NASDAQ100には使わない | 必須 |

現金200万円を信用取引の担保として使う場合、それは完全な機動余力ではない。したがって、200万円は信用担保に算入しない未拘束資金として扱う。

## 7. 最終判定

今回の再判定では、ポートフォリオの主軸を以下へ変更する。

| 旧主軸 | 新主軸 |
|---|---|
| MU/SNDKによるAIメモリ集中 | FANG+積立 + 期限付きMU/SNDK NISA + 信越ポリマー + ソニー + SPCX戦略的オプション |

明確な行動は以下。

1. **信越ポリマーをCore Buyに格上げ**し、初期10%を許容上限、最大20%として扱う。
2. **ソニーをCore Buyに格上げ**し、10-20%を中核候補として扱う。
3. **信越ポリマーとソニーは固定順序で買わず**、安全域、期待収益率、未拘束現金200万円維持を比較して段階取得する。
4. **FANG+は制度内の高効率成長配分枠として継続**し、SPCX採用だけでは積立分割しない。
5. **MU特定3株はイベント前の戦術売却枠**とし、T-5から決済開始、T-3原則完了、T-2最終期限とする。
6. **MU NISA5株とSNDK NISA1株は2026年末を基本期限とする期限付き成長投資**として保有し、延長する場合も恒久保有ではなく新たな期限と条件を設定する。
7. **SPCXは実質合算比率で管理する長期オプション**とし、8%で注意、10%超で直接SPCXの一部利確を検討する。
8. **未拘束の現金・MMF200万円は必須条件**として維持し、信用担保に算入しない。
